# Buscaip
Painel para realizar busca de um ip determinado pelo usuário!

# Modo de uso:

___Linux:___
```bash
apt-get install git && apt-get install jq && apt-get install curl
git clone
cd buscaip
chmod +x painelip.sh
./painelip.sh


```
___Caso o chmod não funcione:___
```bash
bash painelip.sh
```

___Pare Termux:___

```bash

apt install jq

apt install curl
```
